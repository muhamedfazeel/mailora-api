# Mailora API

## Description

Mailora API application. This will be handling the database related operations and core logic

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
